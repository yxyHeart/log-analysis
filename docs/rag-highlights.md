## 完整管线全景

```
                    SearchRequest
                    ┌──────────────────────────────────┐
                    │ query, top_k,                     │
                    │ use_hybrid, use_reranker,         │
                    │ use_query_rewriting, llm_config   │
                    └──────────────┬───────────────────┘
                                   │
                    ┌──────────────▼───────────────────┐
                    │       查询改写 (可选)              │
                    │  query_rewriter.rewrite_and_expand │
                    │  → [original, rewritten, v1, v2]   │
                    └──────────────┬───────────────────┘
                                   │
              ┌────────────────────┼────────────────────┐
              │                    │                    │
     ┌────────▼────────┐ ┌───────▼────────┐ ┌───────▼────────┐
     │  variant 1 检索  │ │  variant 2 检索  │ │  variant N 检索  │
     │ ┌─────────────┐ │ │ ┌─────────────┐ │ │ ┌─────────────┐ │
     │ │  BM25 检索   │ │ │ │  BM25 检索   │ │ │ │  BM25 检索   │ │
     │ │  (jieba分词) │ │ │ │  (jieba分词) │ │ │ │  (jieba分词) │ │
     │ └──────┬──────┘ │ │ └──────┬──────┘ │ │ └──────┬──────┘ │
     │ ┌──────▼──────┐ │ │ ┌──────▼──────┐ │ │ ┌──────▼──────┐ │
     │ │  向量检索    │ │ │ │  向量检索    │ │ │ │  向量检索    │ │
     │ │ (MiniLM编码) │ │ │ │ (MiniLM编码) │ │ │ │ (MiniLM编码) │ │
     │ └──────┬──────┘ │ │ └──────┬──────┘ │ │ └──────┬──────┘ │
     │ ┌──────▼──────┐ │ │ ┌──────▼──────┐ │ │ ┌──────▼──────┐ │
     │ │  RRF 融合    │ │ │ │  RRF 融合    │ │ │ │  RRF 融合    │ │
     │ └──────┬──────┘ │ │ └──────┬──────┘ │ │ └──────┬──────┘ │
     └────────┼────────┘ └───────┼────────┘ └───────┼────────┘
              │                  │                    │
              └──────────────────┼────────────────────┘
                                 │
                    ┌────────────▼────────────────────┐
                    │     合并去重 (all_candidates)     │
                    └────────────┬────────────────────┘
                                 │
                    ┌────────────▼────────────────────┐
                    │    Cross-encoder 重排 (可选)       │
                    │  ms-marco-MiniLM-L-12-v2         │
                    │  top-50 → 精排 → top-k            │
                    └────────────┬────────────────────┘
                                 │
                    ┌────────────▼────────────────────┐
                    │         返回 top-k 结果           │
                    └─────────────────────────────────┘
```

所有阶段通过 `SearchRequest` 参数独立控制：`use_hybrid`、`use_reranker`、`use_query_rewriting`。
